WEB (HTML e CSS)

Resumo

Neste curso você irá aprender como os navegadores funcionam, quais linguagens entendem e como criar sites através do HTML e estilizar com CSS em suas últimas versões.


Prepare-se para descobrir como o mundo WEB funciona.

Requisitos

  • Ter um computador
  • 2 GB RAM
  • Processador i3 5°geração ou semelhante
  • Ter conhecimento sobre informatica básica
  • Possuir conhecimento sobre o pacote ofiice ou semelhante
  • Ter idade de 15 há 24 anos

Conteudo Do Curso

  • HTML
  • Construção de tabelas
  • Utilização de imagens
  • Teoria de encapsulamento
  • Formulários
  • CSS
  • Estilizar por clss e id
  • Estilizar por tag
  • Estilizar imagens
  • Animações
